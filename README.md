### Resumo

Equipes de suporte de TI geralmente dependem de bases de dados de conhecimento que possuem mecanismos de pesquisa tradicionais que frequentemente falham para encontrar os artigos para os problemas pesquisados. E quando encontram, as respostas não são imediatas, fazendo com que os analistas de suporte tenham que ler todo o artigo e muitas vezes múltiplos artigos em sua totalidade para obter as instruções que precisam.

É neste contexto que esta Prova de Conceito se insere visando melhorar a busca pelas instruções e respostas a problemas conhecidos de forma mais ágil, versátil e concisa por meio de um sistema de Geração Aumentada por Recuperação (RAG) utilizando LLMs transformers gratuitos pequenos suficiente para ser possível executá-los localmente em um hardware relativamente modesto sem abrir mão da eficácia.

### Abstract

IT support teams often rely on knowledge databases that use traditional search mechanisms, which frequently fail to find articles addressing the issues being searched. And even when they do, the answers are not immediate, forcing support analysts to read entire articles—and often multiple ones—in full to obtain the instructions they need.

It is within this context that this Proof of Concept is positioned, aiming to improve the search for instructions and responses to known issues in a faster, more versatile, and concise manner. This is achieved through a Retrieval-Augmented Generation (RAG) system using free and lightweight transformer-based LLM models that can be run locally on relatively modest hardware without compromising efficiency.
